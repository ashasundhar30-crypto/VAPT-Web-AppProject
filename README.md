# Vulnerability Assessment & Penetration Testing of a Web Application

## Domain
Cyber Security

## Project Overview
This project demonstrates Vulnerability Assessment and Penetration Testing (VAPT) on an open-source web application (OWASP Juice Shop). The objective is to identify vulnerabilities, exploit them in a controlled environment, and provide mitigation recommendations.

## Tools Used
- Kali Linux
- Nmap
- OWASP ZAP
- Burp Suite

## Methodology
1. Environment Setup
2. Reconnaissance
3. Automated Vulnerability Scanning
4. Manual Penetration Testing
5. Exploitation
6. Risk Analysis
7. Recommendations

## Key Vulnerabilities
- SQL Injection
- Cross Site Scripting (XSS)
- Weak Authentication
- Security Misconfiguration
- Missing Security Headers
